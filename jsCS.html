JavaScript Cheat Sheet

This page contains common JavaScript syntax and examples i came across while learning.

Variables & Data Types

  • let — Block-scoped variable
  • const — Block-scoped constant
  • var — Function-scoped variable (older)
  • typeof — Checks the data type (e.g., typeof 5)

Data Structures

  • Number — e.g., let age = 25;
  • String — e.g., let name = "Alex";
  • Boolean — true or false
  • Array — e.g., let fruits = ["apple", "banana"];
  • Object — e.g., { name: "Sam", age: 30 }
  • null — Intentional empty value
  • undefined — Variable declared but not set

Operators

  • +, -, *, / — Math
  • === — Strict equality
  • &&, ||, ! — Logical operators

Functions

  • function greet() { alert("Hi!"); } — Regular function
  • const add = (a, b) => a + b; — Arrow function
  • return — Returns a value from a function

Control Structures

  • if (x > 10) { ... } — If statement
  • else — Runs when if is false
  • else if — Additional condition
  • switch — Multiple conditions
  • for loop — for (let i = 0; i < 5; i++)
  • while loop — Runs while condition is true

DOM Basics

  • document.getElementById("id")
  • document.querySelector(".class")
  • element.innerText — Get/set text
  • element.innerHTML — Get/set HTML
  • element.style.color = "blue"; — Change style

Events & Debugging

  • element.addEventListener("click", function) — Click event
  • console.log("Hi!") — Logs to console
  • alert("Hello!") — Shows popup
