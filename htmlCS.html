HTML Cheat Sheet

This page contains basic HTML elements and how to use them.

Basic Tags

  • <!DOCTYPE html> — Defines the document type
  • <html> — Root of an HTML page
  • <head> — Contains meta info, title, links
  • <body> — Content goes here

Text Tags

  • <h1> to <h6> — Headings (largest to smallest)
  • <p> — Paragraph
  • <strong> — Bold text
  • <em> — Italic text
  • <br> — Line break

Lists and Links

  • <ul> — Unordered list
  • <ol> — Ordered list
  • <li> — List item
  • <a href="url"> — Link to another page

Images and Media

  • <img src="image.png" alt="desc"> — Display an image
  • <video> and <audio> — Embed media
