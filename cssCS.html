CSS Cheat Sheet

Some common Css properties i learned.

Basic Styling

  • color: — Text color (e.g., color: red;)
  • background-color: — Background color
  • font-size: — Text size
  • font-family: — Font type (e.g., Arial, sans-serif)
  • text-align: — Aligns text (left, center, right)

Box Model

  • margin: — Space outside an element
  • padding: — Space inside an element
  • border: — Adds a border (e.g., 1px solid black)
  • width: — Sets the width of an element
  • height: — Sets the height

Other Useful Properties

  • display: — Layout behavior (e.g., block, inline, none)
  • position: — Controls how an element is placed (e.g., static, relative)
  • text-decoration: — Underline, none, etc.
  • border-radius: — Rounds the corners
  • box-shadow: — Adds a shadow around elements
